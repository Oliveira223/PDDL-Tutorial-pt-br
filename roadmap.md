# Roadmap (Trilha de Aprendizagem)

## Módulo 00 — Fundamentos
- Introdução ao PDDL → objetivo: modelo mental de planejamento
- Paradigma declarativo → objetivo: entender “descrever” vs “programar”

## Módulo 01 — PDDL
- Visão Geral → domínio vs problema
- Estrutura do Domínio → blocos do domínio e ação
- Estrutura do Problema → objetos, init e goal
- Predicados → vocabulário e aridade
- Ações → parâmetros, pré-condições e efeitos
- Pré-condições → aplicabilidade e operadores lógicos
- Efeitos → adição/remoção e consistência de estado
- Tipos → restringir instanciações e clareza de modelagem
- Restrições → regras globais e limites do goal
- Exemplo Completo → domínio + problema integrados

## Módulo 02 — Python
- Visão Python → por que automatizar PDDL
- Leitura PDDL → carregar/inspecionar domínio e problema
- Criação de Domínios → gerar PDDL programaticamente
- Manipulação de Problemas → montar instâncias em lote
- Integração com Planner → executar e coletar planos

## Módulo 03 — Planners
- Visão Planners → o que é um planner e entradas/saídas
- Algoritmos de Busca → base conceitual (ex.: A*)
- Heurísticas → por que acelera e como impacta
- Fast Downward → ferramenta prática
- Debug de Planejamento → metodologia de diagnóstico

## Módulo 04 — Projetos
- Blocksworld → domínio clássico para consolidar ações/efeitos
- Grid Navegação → transições e restrições de movimento
- Logística → modelagem mais realista e composição
- Projeto Final → domínio do aluno, com revisão

## Prática contínua
- Exercícios: `exercicios/01 - Básicos` → `02 - Intermediários` → `03 - Desafios`
- Exemplos: pasta `exemplos/` para referência rápida
